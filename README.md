# KawaK - Meritocratic Writing Platform

## About

KawaK is a free-to-use meritocratic writing tool designed to help writers improve through community feedback.

### Key Features

- **Peer Feedback System**: Users provide constructive feedback on each other's writing
- **Points & Recognition**: Earn points for quality feedback to spotlight your own work
- **Tribunal System**: Fair dispute resolution for users who feel unfairly treated
- **Merit-Based Visibility**: Quality contributors gain platform visibility for their writing

## How It Works

KawaK creates a fair, community-driven ecosystem where writers can improve through genuine peer review while being rewarded for helping others. The more quality feedback you provide, the more opportunity you have to showcase your own writing.
